# Chess Analyzer for macOS

## Welcome to Chess Analyzer!

Chess Analyzer is a powerful desktop application for analyzing chess games from Chess.com. It provides real-time game analysis, move evaluation, and personalized improvement suggestions.

## Installation

1. **Drag the ChessAnalyzer.app to the Applications folder** (or the Applications shortcut on the left)
2. The app is now installed and ready to use!

## Features

- **Game Analysis**: Analyze your Chess.com games with Stockfish engine
- **Move Evaluation**: Get centipawn scores and blunder detection
- **AI Guidance**: Receive personalized improvement suggestions
- **Web Interface**: Modern web-based UI for easy interaction
- **Cross-Platform**: Works on macOS, Windows, and Linux

## Getting Started

1. Launch ChessAnalyzer from your Applications folder or Launchpad
2. The app will open a web interface in your default browser
3. Enter a Chess.com username with public games (like "chesscom")
4. Click "Fetch Games" to analyze their chess games
5. View detailed analysis and improvement suggestions

## Requirements

- macOS 10.14 or later
- Internet connection for Chess.com API access
- No additional installation required - everything is bundled!

## Support

For issues or questions:
- Check the web interface for error messages
- Ensure you're using a Chess.com username with public games
- The app includes comprehensive logging for troubleshooting

## Uninstall

To uninstall Chess Analyzer:
1. Drag ChessAnalyzer.app from Applications to the Trash
2. Empty the Trash
